# Sytem Bath Interaction Algorithm beyond the Lindblad dynamics
This repository contains numerical experiments for system-bath interaction algorithms for dissipative state preparation beyond the leading order Lindblad approximation.

The code supports numerical studies in main text of https://arxiv.org/pdf/2512.03457 for three representative many-body models:

- Transverse-field Ising model (TFIM)
- Fermi-Hubbard model
- Axial next-nearest-neighbor Ising model (ANNNI)

The main goal is to study the behavior of the full induced system-bath interaction channel, including fidelity, energy dynamics, and spectral-gap diagnostics for thermal and ground state preparation.

## Numerical experiments

### Transverse-field Ising model

We consider the one-dimensional transverse-field Ising model

$$
H = -J\sum_{i=1}^{L-1} Z_i Z_{i+1} - g\sum_{i=1}^{L} X_i .
$$

The notebook `tfim_thermal_dynamics.ipynb` studies thermal state preparation and plots the fidelity dynamics under repeated system-bath interaction steps.

### Fermi-Hubbard model
We consider the one-dimensional Fermi-Hubbard model

$$
H = -t\sum_{j=1}^{L-1} \sum_{\sigma\in\{\uparrow,\downarrow\}} c_{j,\sigma}^\dagger c_{j+1,\sigma}+U\sum_{j=1}^L (n_{j,\uparrow}-\frac{1}{2})(n_{j,\downarrow}-\frac{1}{2})\,.
$$

The notebook `hubbard_thermal_spectral_gap.ipynb` studies the induced quantum channel for the thermal state preparation and plots the spectral gap of the channel as a function of the coupling strength.

### ANNNI model

We consider the one-dimensional axial next-nearest-neighbor Ising (ANNNI) model

$$
H
=\frac{J_1}{4} \sum_{i} Z_i Z_{i+1}
+\frac{J_2}{4} \sum_{i} Z_i Z_{i+2}
-\frac{G}{2} \sum_{i} X_i\,.
$$

The notebook `annni_ground_dynamics.ipynb` studies ground state preparation for the axial next-nearest-neighbor Ising model. The output figure tracks the energy dynamics during the dissipative preparation process.

## Requirements

* `scipy`

* `quimb`

## Running the notebooks

Clone the repository:

```bash
git clone https://github.com/kewang-math/Interaction-BeyondLindblad.git
cd Interaction-BeyondLindblad
```

Install the dependencies:

```bash
pip install numpy scipy matplotlib tqdm quimb jupyter
```

Then run one of the notebooks:

```text
tfim_thermal_dynamics.ipynb
hubbard_thermal_spectral_gap.ipynb
annni_ground_dynamics.ipynb
```

The generated figures are saved in the `figures/` folder.
